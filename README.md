# Demo web page

This page is a demo for a webpage to host your project links.  You can use this to link your portfolio or links such as to your project code.
1. Replace the title with your own     
2. Format an image that is 1000 wide by 500 high in pixels and replace this in the repo.  
3. Update the src = "link"
4. Replace the paragraph with your own
5. update the name
6. update the email
7. add or subtract links as needed.
